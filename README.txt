CIVILINE — Civil Engineering NPTEL Knowledge Hub
=================================================

WHAT'S IN THIS ZIP
-------------------
- index.html              -> Front page, links to all 10 subjects
- video-library.html       -> All NPTEL lecture links in one place
- subjects/*.html          -> 10 subject pages, each with:
                              - a labelled diagram (SVG)
                              - a link to the official NPTEL lecture series
                              - 8 original MCQ practice questions with
                                instant answer + explanation on click
- assets/css/style.css     -> Shared "blueprint" theme styling
- assets/js/quiz.js        -> Shared quiz interaction logic
- assets/images/*.svg      -> 10 original diagrams (one per subject)

HOW TO USE
-----------
Unzip anywhere and open index.html in any browser. No server or
installation needed — it's a fully static site.

ABOUT THE VIDEOS
------------------
This package cannot include actual video FILES — video generation
isn't something that can be produced here, and re-hosting NPTEL's
copyrighted video content would not be appropriate.

Instead, every subject page (and video-library.html) links directly
to that subject's OFFICIAL NPTEL course page on nptel.ac.in, where
the real recorded lecture series is hosted for free. These links
were verified against NPTEL's own site at the time this site was
built. If NPTEL restructures a course URL in the future, search
nptel.ac.in for the course title to find its new location.

ABOUT THE QUESTIONS
---------------------
The 80 MCQs are original practice questions written to cover the
standard topics taught in each subject's NPTEL course — they are
not copied from any NPTEL exam or question bank. They're meant for
self-study revision, not as a substitute for the official course
assignments or certification exam.
