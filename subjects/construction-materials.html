# Construction Management – Complete Important Points & Formula Sheet (Civil Engineering Exam Revision) This is a high-value revision sheet covering the most important concepts and formulas frequently asked in diploma, B.E./B.Tech., and competitive exams. --- # 1. Definition Construction Management is the process of **planning, organizing, scheduling, directing, coordinating, and controlling** construction activities to complete a project: * On Time * Within Budget * With Required Quality * Safely --- # 2. Objectives * Complete project on schedule * Minimize project cost * Improve quality * Increase productivity * Reduce waste * Ensure safety * Customer satisfaction --- # 3. Functions of Management **PODSC** * Planning * Organizing * Directing * Staffing * Controlling --- # 4. Project Life Cycle 1. Feasibility Study 2. Planning 3. Design 4. Tendering 5. Construction 6. Monitoring 7. Commissioning 8. Handover 9. Maintenance --- # 5. Project Triangle ``` TIME / \ / \ QUALITY----COST ``` If one changes, the others are usually affected. --- # 6. Work Breakdown Structure (WBS) Divide project into smaller activities. Example: Building ↓ Foundation ↓ Columns ↓ Beams ↓ Slab ↓ Brickwork ↓ Plaster ↓ Painting --- # 7. Scheduling Methods * Bar Chart * Gantt Chart * Milestone Chart * CPM * PERT * Line of Balance --- # 8. Bar Chart Advantages * Simple * Easy to prepare * Good for small projects Disadvantages * Cannot show dependency * Difficult for large projects --- # 9. Network Terminology Activity Task consuming time and resources. Event Beginning or completion of activity. Dummy Activity Consumes no time or resources. Critical Activity Activity having zero float. --- # 10. CPM (Critical Path Method) Used when activity duration is known. Features * Determines Critical Path * Calculates Float * Controls Project Time Formula ``` Project Duration = Sum of Critical Path Activities ``` Critical Path Longest path in network. --- # 11. Forward Pass Formula ``` Earliest Start (ES) = Maximum EF of predecessor ``` ``` Earliest Finish EF = ES + Duration ``` --- # 12. Backward Pass Formula ``` Latest Finish LF = Minimum LS of successor ``` ``` Latest Start LS = LF – Duration ``` --- # 13. Float Formula Total Float ``` TF = LS − ES ``` or ``` TF = LF − EF ``` Free Float ``` FF = ES(successor) − EF ``` --- # 14. PERT Formula Expected Time ``` TE = (O + 4M + P)/6 ``` Where O = Optimistic M = Most Likely P = Pessimistic Variance ``` σ² = ((P − O)/6)² ``` Standard Deviation ``` σ = (P − O)/6 ``` --- # 15. Resource Types * Labour * Material * Machinery * Money * Time --- # 16. Labour Productivity Formula ``` Productivity = Output / Labour Hours ``` Example 100 m² plaster 20 labour-hours Productivity =5 m²/hr --- # 17. Labour Efficiency ``` Efficiency % = Actual Output ÷ Standard Output × 100 ``` --- # 18. Equipment Productivity ``` Productivity = Quantity / Working Time ``` --- # 19. Equipment Utilization ``` Utilization % = Working Hours ÷ Available Hours × 100 ``` --- # 20. Material Wastage ``` Wastage % = Waste ÷ Total Material × 100 ``` --- # 21. Inventory Terms EOQ Economic Order Quantity ABC Analysis A → Expensive items B → Medium value C → Low value FIFO First In First Out LIFO Last In First Out --- # 22. Cost Components * Material * Labour * Equipment * Overhead * Profit * Taxes --- # 23. Cost Variance ``` CV = Budgeted Cost − Actual Cost ``` Positive ↓ Cost Saving Negative ↓ Cost Overrun --- # 24. Schedule Variance ``` SV = Planned Progress − Actual Progress ``` --- # 25. Earned Value Terms PV Planned Value EV Earned Value AC Actual Cost --- # 26. Earned Value Formulas Cost Performance Index ``` CPI = EV / AC ``` Schedule Performance Index ``` SPI = EV / PV ``` Interpretation CPI >1 ↓ Project under budget SPI >1 ↓ Ahead of schedule --- # 27. Quality Management Quality Assurance (QA) Prevent defects Quality Control (QC) Find defects --- # 28. Concrete Quality Tests * Slump Test * Cube Test * Compaction Factor * Rebound Hammer * UPV Test --- # 29. Safety Equipment * Helmet * Gloves * Shoes * Goggles * Ear Protection * Reflective Jacket * Safety Belt --- # 30.
